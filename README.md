# Nuxt > Basic example

Afin de testay le HMR sur une base saine.

## Usual commands

```bash
cd tastytest

# ? / export NODE_OPTIONS=--openssl-legacy-provider
yarn dev
# http://localhost:3000 
```

## Installation

```bash
yarn create nuxt-app tastytest
# > tout par defaut
```
